# react-native-damoov

React Native implementation of Damoov's Telematics SDK

## Installation

```sh
npm install react-native-damoov
```

## Usage

```js
import Damoov from "react-native-damoov";

// ...

const result = await Damoov.multiply(3, 7);
```
